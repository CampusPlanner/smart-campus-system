Project Overview

Welcome to the Smart Campus System! This project, developed by students at [Your College/University Name], aims to improve campus life with real-time communication and automation.

Features include:

  • Campus Help Bot
  • Event Reminders and Notifications
  • Peer Networking and Study Buddy Matching
  • Emergency Support System
  • Campus Essentials Locator
  • Student Projects Showcase

This system uses WhatsApp Business API to make campus communication easier and more efficient.

How it Works

Students can interact with various campus services through WhatsApp, receive real-time updates about events, connect with study buddies, find campus essentials, and even get emergency assistance.
